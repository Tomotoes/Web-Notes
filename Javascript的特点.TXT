【1】简单性
它是基于Java基本语句和控制流之上的简单而紧凑的设计，是学习Java的好过渡，而且，它的变量类型是采用弱类型，未采用严格的数据类型。

【2】安全性
Javascript 不允许访问本地硬盘，不能将数据存入到服务器上，不允许对网络文档进行修改和删除，只能通过游览器 实现信息游览或 动态交互，从而有效的防止数据的丢失。

【3】动态性
Javascript 可以直接对用户或客户 输入做出响应，无须经过 web程序。
它对用户的响应采用以事件驱动的方式进行，即由某种操作动作 引起相应的事件响应，
如：点击鼠标，移动窗口，选择菜单等。

【4】跨平台性
Javascript 依赖于 游览器 本身，于操作环境无关。只要能运行游览器的计算机，并安装了支持 Javascript的游览器 就可以正确执行，从而实现了"编写一次，走遍世界"的梦想。
